Launch Project

World clock application that displays the local time of different cities around the world. The application is able to detect the local timezone of the user and display the appropriate time. Weather Project

Launch Project

Weather Application inspired by the popular anime "ONE PIECE". Fully functionial weather application capable of displaying the current and future weather forecast.
